## üé® The 6-Phase Process / Le Processus en 6 Phases

### Phase 1: Clarification (The Agent Asks 6 Questions)

The agent will ask about:
1. **Product Type** - What exactly are you creating?
2. **Marketplace** - Where will you sell it?
3. **Target Audience** - Who will buy it?
4. **Design Style** - What aesthetic?
5. **Tools** - What software will you use?
6. **Constraints** - Any limitations?

**Your task:** Answer clearly and honestly for best results.

### Phase 2: Creative Directions (You Get 2-4 Options)

The agent proposes multiple creative directions, each with:
- Name and description
- Visual style references
- Color palette (with hex codes)
- Typography recommendations
- Layout concepts

**Your task:** Review and pick your favorite, or ask for variations.

### Phase 3: Detailed Specifications

The agent develops your chosen direction with:
- Detailed layout specifications
- Color roles and combinations
- Typography hierarchy
- Visual elements guide
- If templates: list all variations

**Your task:** Review and provide feedback or approval.

### Phase 4: Technical Requirements

Complete specifications including:
- Exact dimensions (pixels or cm + DPI)
- File formats needed
- Folder structure
- File naming convention
- Technical notes

**Your task:** Download and save this information for designers.

### Phase 5: Sales Optimization

Marketplace-ready content:
- 3-5 SEO-optimized titles
- Long product description
- 10-20 relevant tags/keywords
- 3-5 sales benefits
- Mockup recommendations

**Your task:** Copy this content to your marketplace listings.

### Phase 6: Iterative Refinement

The agent suggests:
- Design variations
- Upsell opportunities
- Bundle suggestions
- Seasonal adaptations

**Your task:** Request improvements or additional concepts.

## ‚ùì Common Questions / Questions Courantes

**Q: How long does the process take?**
A: Usually 2-4 hours from start to complete specifications. Actual design work depends on your skills and software.

**Q: Do I need design experience?**
A: No! The agent explains everything. If you use Canva, it's very beginner-friendly.

**Q: Can I use these templates on multiple platforms?**
A: Yes! The agent creates adaptable designs. One design can work across Etsy, Creative Market, Gumroad, etc.

**Q: How specific should my brief be?**
A: As specific as possible. More details = better results. Include target audience, use case, style preferences, timeline.

**Q: Can the agent modify its focus?**
A: Yes! Tell it if you want to focus on SEO optimization, file organization, brand consistency, or sales copy.

**Q: What if I don't like a direction?**
A: Ask for alternatives. The agent can pivot to different styles, audiences, or approaches.
